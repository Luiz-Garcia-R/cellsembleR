# cellsembleR

<!-- badges: start -->

<!-- badges: end -->

**cellsembleR** is an R package designed to facilitate the extraction,
analysis, and visualization of biological signatures from bulk
transcriptomic data.

The package provides a unified framework for estimating cellular
signatures using reference-based methods (e.g., EPIC, xCell,
MCP-counter) or user-defined gene signatures, followed by statistical
comparison, association analysis, dimensionality reduction, and
visualization.

Unlike traditional deconvolution-only approaches, **cellsembleR**
focuses on turning signature scores into interpretable biological
features that can be compared across experimental conditions and
integrated into downstream analyses.

## Installation

You can install the development version directly from GitHub:

``` r
# Install remotes if not already installed
install.packages("remotes")

# Install cellsembleR from GitHub
remotes::install_github("Luiz-Garcia-R/cellsembleR")
```

## Concept Overview

Bulk RNA-seq profiles represent the combined transcriptional activity of
multiple cell populations and biological programs. cellsembleR provides
tools to estimate and analyze these hidden biological features.

The general workflow is:

1.  Bulk RNA-seq data
2.  Cell signature estimation: (EPIC / xCell / MCP-counter / Custom
    signatures)
3.  cellsemble_result object

-   Statistical analysis (compare/delta)
-   Visualization (heatmap/dimred/boxplot)

4.  Biological interpretation

## Input Data Format

cellsembleR works with rna_project objects generated by autornaR.

A valid project should contain:

1.  Expression matrix

-   Gene expression values
-   Samples as columns
-   Gene identifiers as rows

Example:

| Gene | Sample1 | Sample2 | Sample3 | Sample4 |
|------|---------|---------|---------|---------|
| FAP  | 12.5    | 10.2    | 35.4    | 40.1    |
| LUM  | 20.1    | 18.4    | 50.3    | 55.2    |
| CD3D | 5.2     | 8.1     | 2.0     | 3.5     |

2.  Sample metadata

The metadata should contain:

-   Sample: matching sample identifiers
-   Group: experimental condition

Example:

| Sample  | Group     |
|---------|-----------|
| Sample1 | Control   |
| Sample2 | Control   |
| Sample3 | Treatment |
| Sample4 | Treatment |

## Main Workflow

### Estimate cellular signatures

cellsembleR provides multiple approaches:

-   cellsemble_epic_run(): Estimate immune and stromal cell fractions
    using EPIC.
-   cellsemble_xcell_run(): Estimate a broad landscape of immune and
    stromal signatures using xCell.
-   cellsemble_mcp_run(): Estimate cellular populations using
    MCP-counter.
-   cellsemble_custom_run(): Calculate user-defined biological
    signatures from custom gene lists.

Example:

``` r

# EPIC-based signatures
res <- cellsemble_epic_run(
    project = my_project
)

# Custom biological signatures
res <- cellsemble_custom_run(
    project = my_project,
    gene_sets = signatures
)
```

## Signature Exploration and Statistics

After generating a cellsemble_result object:

### Compare signatures between groups

``` r

comparison <- cellsemble_compare(
    res
)
```

Performs statistical testing between two groups, including:

-   automatic test selection;
-   effect size estimation;
-   bootstrap confidence intervals;
-   multiple testing correction.

### Visualize signature differences

``` r

cellsemble_delta(
    comparison
)
```

Creates an effect-size based visualization highlighting biological
programs enriched in each group.

### Explore individual signatures

``` r

cellsemble_boxplot(
    res,
    signature = "T_cells"
)
```

Visualizes signature distributions across experimental groups.

### Signature Relationships

``` r

cellsemble_association(
    res
)
```

Supports:

-   Pearson correlation;
-   Spearman correlation;
-   signature-signature relationships;
-   optional gene-level associations.

### Global Signature Landscape

``` r

cellsemble_dimred(
    res,
    method = "umap"
)
```

Visualizes sample relationships based on inferred biological signatures.

Available methods:

-   PCA
-   UMAP

### Heatmaps

``` r

cellsemble_heatmap(
    res
)
```

Provides:

-   sample-level signature profiles;
-   group-level aggregated profiles;
-   hierarchical clustering;
-   biological pattern visualization.

### Custom Biological Signatures

One of the main features of cellsembleR is the ability to evaluate
user-defined biological programs.

Example:

``` r

cells_signatures <- list(

# Parent population
Lymphocytes = c(
"CD3D",
"CD3E",
"CD79A",
"MS4A1"
),

# Child population
T_cells = c(
"CD3D",
"CD3E",
"TRBC1"
)
)

res <- cellsemble_custom_run(
    project = my_project,
    signatures = cells_signature
)
```

Parent-child relationships can also be modeled:

``` r

parent_sets <- list(
    T_cells = "Lymphocytes"
)

res <- cellsemble_custom_run(
    project = my_project,
    signatures = cells_signatures,
    parent_sets = parent_sets,
    adjustment = "subtract"
)
```

This enables conditional interpretation of biological programs by
accounting for broader cellular identities.

## Output Objects

Most functions return structured S3 objects:

-   cellsemble_result
-   cellsemble_compare
-   cellsemble_delta
-   cellsemble_corr
-   cellsemble_dimred
-   cellsemble_heatmap
-   cellsemble_boxplot

These objects store both results and analysis parameters, facilitating
reproducibility and downstream exploration.

## Example Workflow

``` r

# Estimate cellular signatures
res <- cellsemble_epic_run(
    project = my_project
)

# Compare groups
comp <- cellsemble_compare(
    res
)

# Visualize differences
cellsemble_delta(
    comp
)

# Heatmap visualization
cellsemble_heatmap(
    res
)

# Dimensional reduction
cellsemble_dimred(
    res,
    method = "pca"
)
```
